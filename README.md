# ChantMagick

## 概要
即興で詠唱をして魔法を使うシステムです。

## install
以下のライブラリを環境にインストールしてください。

```
Ginza
Flask
sklearn 
csv
numpy
```
Ginzaは以下のコマンドでインストールしてください。
```
pip install "https://github.com/megagonlabs/ginza/releases/download/latest/ginza-latest.tar.gz"
```
